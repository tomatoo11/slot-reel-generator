# パチスロ出目画像サイト

依存なしでそのまま開ける静的サイトです。  
図柄画像を置いて `index.html` をブラウザで開くと、筐体画像の白いリール窓に出目を合成してPNG保存ができます。筐体画像と図柄画像は画面から直接アップロードできます。

## ファイル構成

- `index.html`: UI本体
- `styles.css`: レイアウトと見た目
- `app.js`: 出目選択、Canvas合成、PNG保存
- `machineConfig.js`: 機種設定、図柄一覧、素材パス、リール窓座標

## 画像の置き場所

以下のパスに画像を置いてください。

- `assets/symbols/bell.png`
- `assets/symbols/cherry.png`
- `assets/symbols/replay.png`
- `assets/symbols/seven.png`
- `assets/symbols/bar.png`
- `assets/symbols/grape.png`

筐体画像は画面のファイル選択から直接読み込めます。設定ファイルの `machine-body.png` 指定を使うこともできます。  
図柄画像も画面の各ファイル選択から直接読み込めます。未配置だとプレースホルダー表示になります。

## カスタマイズ

`machineConfig.js` を編集すると次を変更できます。

- 機種名
- キャンバスサイズ
- 図柄の種類
- 各リールの初期値
- 筐体画像のパス
- 各リール窓の位置、幅、高さ
- 図柄の上下に見せる並び順

図柄追加時は `symbols` に追加し、必要なら `reels` の初期値も更新してください。

## 使い方

1. `index.html` をブラウザで開く
2. 画面の「筐体画像」からパチスロ機画像を選ぶ
3. 各リールの図柄パネルをクリックして停止図柄を選ぶ
4. プレビュー上の黄色い枠をドラッグして白いリール窓に合わせる
5. 右下ハンドルまたはスライダーでサイズを微調整する
6. `PNGを保存` を押す

現在の初期シンボルは `剣 / ドリンク / 守 / 面 / トマト / スライム / 青七` です。

## 補足

- 今回の初期設定は、提供された筐体画像に合わせた3リール表示です
- もともとの設計は設定ファイル分離済みなので、別画像や別座標へ差し替えやすい構成です
- 座標は `machineConfig.js` を直接編集しなくても、画面上の枠ドラッグとスライダーで直感的に調整できます
- 将来的に複数機種対応したい場合は、設定ファイルを複数持つ構成に広げやすいよう分離しています
- ローカルファイルとして開いて画像が読めないブラウザがある場合は、簡易HTTPサーバーで配信してください

# コミュニティ共有ガイド

このフォルダは、そのまま GitHub に上げれば公開できます。

## すぐ配る方法

`pachislot-generator-share.zip` をそのまま配ってください。  
受け取った人は展開して、`index.html` を開けば使えます。

## GitHub に上げる方法

1. GitHub で新しい `Public` リポジトリを作る
2. リポジトリ画面の `uploading an existing file` を押す
3. このフォルダの中身をまとめてドラッグする
4. 下の `Commit changes` を押す
5. `Settings` → `Pages` を開く
6. `Deploy from a branch` を選ぶ
7. `main` と `/root` を選んで `Save`

数分すると公開URLが出ます。

## GitHub に上げるときに入れるもの

- `index.html`
- `app.js`
- `machineConfig.js`
- `styles.css`
- `assets` フォルダ

## 入れなくていいもの

- `serve.log`
- `serve.err.log`
- `index_utf8_dump.txt`

## 公開後の使い方

- 左の絵柄を選ぶ
- 右の筐体プレビューで確認する
- 必要なら画像をコピー、またはスクリーンショットする

## 補足

このアプリは静的サイトです。  
サーバー側のプログラムは不要なので、GitHub Pages と相性がいいです。
